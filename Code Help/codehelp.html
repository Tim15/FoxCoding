Basics

Js is a coding language used to make the data that you see on websites interactive. The coding language js is for creating an interaction
with content on a website and you.

One of the first things that you should learn about in js are the data types. They are the String, Number, Boolean, Array, and Object
String: A string is a series of characters within quotes, Example: "Lorem ipsum dolor sit amet."
Number: A number is any number with or without a decimal, Example: 5
Boolean: A boolean is a value that is only true or false, Example: true
Array: An array is values put in between square brackets, Example: ["hi", "bye", "die"]
Object: An object is a value with multiple properties, an Example: {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"}
