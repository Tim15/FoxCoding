HTML



Intro

Html stands for Hyper-Text Markup language, and is a language that is used to create the content of a web page, like this paragraph.
This language uses CSS to style it and make it look good. Also Html uses JavaScript to make it interactive. Html is made up of tags, which can be opening, or closing tags. The way you write opening tags is <put the tag name and attributes here>, and this is an example of an ending tag </put the tag name here>. For example there is the p tag for paragraphs, which is <p> and </p>. Before we get started you need to learn some tags to input js, and css. For Js do <script> and </script> then put the Js inside. For css do <style> and </style>.
