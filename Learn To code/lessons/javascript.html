Javascript



Intro

Javascript is a programming language that is used to add interactivity to html, such as Making things happen when you push a button, or doing math.
This language is used with html. Javascript is made up of many different things, that we will go over in the tutorial.

Note:

  • Java script is case sensitive, so if you name something with a capital, when you acess it use the capital
  • Javascript statemnts are usually ended with a semicolon, So if you create a varible, or somthing else, rnd it with a semicolon.

Basics



The first thing you should learn about are variables, Variable store data that can be used later. They are created using the word var, then the one word variable name, = the variable value. You can also make multiple variables in one statement, by adding a comma in between them. Variables can be set to many things, including html elements, and functions, but there are a few basic data types, they can be set to.

  • Strings: Strings are pieces of text, made by surrounding the data in quotes EX: 'This is a string'
  • Numbers: Numbers are what they sound like, numbers. Made by typing a number EX: 1
  • Boleans: Booleans are pieces of data that can only be true, or false, made by typing true, or false EX: true
  • Boleans: Booleans are variables, that have a list of values, made by brackets, and in between them a values then a comma EX: [true, 1, 'This is a string', ['Hi', 'Bye']]
  • Objects: Objects, are variables, that have many attributes, even other objects. made by typing curly braces, and in between them are the attributes, a colon , and values EX: {string: 'This is a string', number: 1, boolean: true, array: [true, 1, 'This is a string'] }
Example:

    Input:

var MyString = 'This is a string variable', MyNumber = 1, MyBoolean = true;

    Output:

A variable called, MyString with a value of 'This is a string variable', MyNumber with a value of 1, and MyBoolean with a value of true.
